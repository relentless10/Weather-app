

🌤️ Weather App
A clean, glassmorphism-styled weather app built with vanilla HTML, CSS and JavaScript. Type in any city and get the current weather conditions back in seconds.
🚀 What It Does
You type a city name, hit submit, and boom — you get back:
🏙️ City name
🌡️ Feels like temperature
☁️ Current conditions
💧 Humidity
🕐 Local time
🌀 Weather icon type
All pulled live from the Visual Crossing Weather API.
🛠️ Built With
HTML
CSS (glassmorphism design)
Vanilla JavaScript (async/await)
Visual Crossing Weather API
⚡ How To Run It
No installs, no frameworks, no drama.
Clone the repo
Open index.html in your browser
Type a city and hit Submit
That's it 🎉

🔑 API Key
This app uses the Visual Crossing Weather API. If you want to run it yourself, grab a free API key at visualcrossing.com and replace the key in app.js.
⚠️ Don't push your API key to GitHub. Look into environment variables when you move to a backend.
